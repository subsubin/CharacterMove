[ CharacterMove_Guide ]

⊙ 분류 : JAVA 심화 방과후학교 프로젝트
⊙ 기간 : 2018.01.12
⊙ 제목 : CharacterMove
⊙ 개발환경 : JAVA (개인)
⊙ 작품설명 :
   1. JAVA 파일을 구별하기 쉽게 4가지 패키지로 구분하여 저장하였다.
     - Constant에서는 아이템과 캐릭터의 크기를 설정하였다.
     - Controller에서는 게임을 실행할 수 있게 구현하였다.
     - Handlers에서는 캐릭터를 키보드로 움직이는 것과 아이템들을 구현하였다.
     - Views에서는 보여지는 캐릭터와 화면을 주로 다루었다.
   2. 게임의 효과음, 배경음악을 설정하였다.
